# Bank Account Management System

## Overview

This project demonstrates a simple Bank Account Management System implemented in **Java**. The application allows users to create a bank account, deposit funds, withdraw funds, and check the current account balance. The project also includes a comprehensive set of **JUnit test cases** to verify the correctness of the implementation.

## Features

- Create a bank account with an initial balance.
- Prevent creation of accounts with negative balances.
- Deposit money into an account.
- Validate deposit amounts to ensure they are positive.
- Withdraw money from an account.
- Prevent withdrawals that exceed the available balance.
- Retrieve the current account balance.
- Unit testing using JUnit.
